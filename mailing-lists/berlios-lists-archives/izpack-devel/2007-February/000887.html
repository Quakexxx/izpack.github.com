A plain text GUI is a pretty though project. Either you get a very
simple line-based interface or you need to stick to some native
libraries for proper terminal support. Java was just not designed for
this.

I researched this some time ago for our open-EIS management tools and
then stuck to line-based interface only (which is pretty cumbersome in
some places like multi-select boxes). But IIRC there are some
swing-wrappers which provide text-mode GUIs.
