Is there a way to generated an automated script when using the
SimpleFinishPanel?

What I have in mind is that users would see the SimpleFinishPanel (and
not the FinishPanel which is irrelevant for 90% of them) but the
documentation would specify how to generate an automated script. For
example:
java -Dautomatedscript=automated.xml -jar my-app-installer.jar.

This would make the SimpleFinishPanel to write the script (eventually
adding a label saying "Automated script generated in:
path/path/automated.xml").

6 and Vista?
