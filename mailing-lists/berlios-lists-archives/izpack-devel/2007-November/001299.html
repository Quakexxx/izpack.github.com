Hi everyone,

IzPack 3.11.0-beta1 is now available for download from
ftp://ftp.berlios.de/pub/izpack/IzPack-install-3.11.0-beta1.jar

This is not an official release. It is basically a snapshot of the SVN
trunk. It should however be very fine for production quality
installers.

We would appreciate if you could test it and report issues so that we
get a good level of quality for what will be the official 3.11.0
release.

In particular, we are currently improving / revamping the
documentation, so do not hesitate to:
- propose enhacements
- clarify some sections
- bring new content
- check if feature X or feature Y is properly documented (we have some
undocumented stuff...).

Cheers
