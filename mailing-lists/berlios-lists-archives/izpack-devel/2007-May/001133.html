Hi, I am working on a few linux shortcut fixes right now. I haven't 
tried Efty Edge, but will do these days. In the mean time if you want to 
debug, simply set up a new debug configuration in eclipse (or whatever 
IDE) and use com.izforge.izpack.installer.Installer as the entry point 
class (put the installer jar and izpack libs in the classpath) and it 
should work.

Are the desktop shortcuts created correctly? Are you running as root or 
normal user (try both)? Gnome/KDE version?

BTW, there is a known bug in some recent Gnome - the directory link 
shortcuts do not appear in the gnome menu. If you have such type of 
shortcuts you should replace them with a browser launch or gnome-opener 
launcher (ping me if you need details).
