Hi Sujay,
to add a feature to a existent panel is simple.
See in the docu how to create an own panel. Often you can derive from
an existent panel and add the needed things. 
But printing is not so simple...
In the mean time Java supports printing, but not on all plattforms in
the manner you mean.
A secound way can be to delegate the printing to an externel process. If
you are
certain that e.g. lpr on all Unix machines...
A third way is to delegate the printing to the user (installer); for
this you can 
add a MultiLineLabel where the path of the license file will be written.
