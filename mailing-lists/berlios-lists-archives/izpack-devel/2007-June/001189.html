Instead of using the os-attribute, you’ll have to add an os element
 

<pack id=”test” name=”testpack”>
 

   <os name=”macos” />
 

</pack>
 

  
 

The os-element also supports a family-attribute.
 

  
 

A much more flexible approach is to use the conditions which can
be defined in a resource called conditions.xml. See the examples and doc 
 

on this mailing list.
