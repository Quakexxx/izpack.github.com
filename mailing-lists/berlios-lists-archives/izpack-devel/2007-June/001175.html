Hi everyone,

I have added support for the Nimbus L&F in the trunk.

This is a nice L&F, still in beta, which is supported / endorsed by
Sun (https://nimbus.dev.java.net/).

You can test it using the 'nimbus' key for the guiprefs laf param of
your installer files. Please note that it requires Java 6 to run. I
had some issues on Mac OS X with Apple Java 6 which is known to be ...
quite unfinished ;-)

Cheers
