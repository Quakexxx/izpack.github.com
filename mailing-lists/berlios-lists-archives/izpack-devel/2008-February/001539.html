I don't think we should drop 1.4 yet. Even though Java 6 is already
there, I suppose there's a lot of 1.4 still around, especially in server
environments. There are even platforms where there is not 1.5 available
yet.

It's not too hard to find Java 1.5 dependencies - just install a 1.4
JDK, use it for compiling and running the installer. It's easy to do in
Eclipse - just configure another JRE and set this as the project's JRE.
