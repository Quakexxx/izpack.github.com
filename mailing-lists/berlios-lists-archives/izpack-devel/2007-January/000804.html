Hi everybody,

I am pleased to announce that the first release candidate of the
IzPack installer version 3.10.0 is available for testing. Please
report any issue to the izpack-devel mailing-list.

It can be downloaded from our public BerliOS FTP space:
ftp://ftp.berlios.de/pub/izpack/IzPack-install-3.10.0-RC1.jar

The source code can be checked-out from:
http://svn.berlios.de/svnroot/repos/izpack/izpack-src/tags/3.10.0-RC1/

Also, a branch holding the development of the 3.10.x versions is available from:
http://svn.berlios.de/svnroot/repos/izpack/izpack-src/branches/3.10/

Cheers
