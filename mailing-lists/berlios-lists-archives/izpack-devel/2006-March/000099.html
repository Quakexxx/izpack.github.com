Hi Hal,
it has setted registry keys during installation...
I should be only an example what todo to write into the registry.
Look the differences between IzPack-install.xml and
IzPack-install-reg.xml in %IzHome%\src\dist-files.
Unfortunately to the really diffs there are different formatings
in the files.

Look into the Registry
HKLM\SOFTWARE\IzForge\IzPack
There is a key named
IzPack 3.8.1 (build 2006.01.06)
It contains some values of different types and some keys.
To demonstrate the locale support some are in Kyrillic.

Start Windows software manager. There is an entry for IzPack.

Start the installation a secound time. With the first panel
you get a warning that there is already an IzPack installation
on your box.

That's all.

Klaus
