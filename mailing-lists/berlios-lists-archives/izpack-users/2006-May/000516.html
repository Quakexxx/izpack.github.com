No, the List contains Packs. Use

  Iterator iter = idata.selectedPacks.iterator();
  while (iter.hasNext())
  {
    String currentPack = ((Pack) iter.next()).name;
    doWhatYouNeed(currentPack);
  }

  This code snipet is from AntInstallerLister.afterPacks.
