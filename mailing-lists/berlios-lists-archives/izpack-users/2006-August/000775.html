Hi Zwirek,
if you use the standalone-compiler pol.xml is in the standalone-compiler.
Else the files are in
%IZPACK_HOME%\bin\langpacks\installer
You can push them into the standalone-compiler with a new build of
IzPack.
Alternatively you can create a custom language file and declare 
the message to be changed there. See docu 

Creating Your Own Panels->The Internationalization of custom panels

Using the custom language file prevent for synchronize at update 
IzPack.

Cheers

Klaus
