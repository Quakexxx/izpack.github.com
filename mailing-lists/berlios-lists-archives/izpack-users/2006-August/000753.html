Hi,
oh boys, there was something, I remember...
May be you need the IZPACK_HOME environment variable.

I have always a batch file or a super ant script to pack the
installations. It contains

set IZPACK_HOME=N:\home\bartzkau\work\xt150_forIzPack\izpack-src

With eclipse I use the VM argument
"-DIZPACK_HOME=n:\home\bartzkau\work\xt150_forIzPack\izpack-src"

The dll have to be as
%IZPACK_HOME%\bin\native\izpack\ShellLink.dll
not else where.
The reference for the dll in install.xml have to be
	</native>
	<native type="izpack" name="ShellLink.dll">
		<os family="windows"/>
	</native>
also not other.

Hope it helps

Klaus
