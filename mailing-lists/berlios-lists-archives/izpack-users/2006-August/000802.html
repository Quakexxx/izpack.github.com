Hello,

I've found a way for creating subgroups in the shortcut-section.

In the documentation for the Desktop Shortcuts (node6.html) is an attribute missing.

The attribute is 'subgroup' for creating a subgroup for the shortcuts.

I've wondered while traversing the sourcecode for the ShortcutData class, because there was a String subgroup, but no entry in the documentation.

So I've tested a little bit and surprise surprise, using this tag with the name of the subgroup creates a folder with this name in the shortcut-menue.


So I can say, that the other mails which relates on the question for the shortcuts written by me should be abolished. Because this is the SOLUTION!!!

So, please add this to the documentation for the next release :-).
