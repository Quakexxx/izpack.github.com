Hi, I guess that this would work with an <executable> element in the <pack> 
element which is placed in a <packs>.

The exact description of it is:

-------------begin--------------
<executable> - mark file executable or execute it

The <executable> tag is a very useful thing if you need to execute something 
during the installation process. It can also be used to set the executable 
flag on Unix-like systems.
--------------end---------------

The best you could do is to read the documentation.
