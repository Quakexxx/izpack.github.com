Hi,

After entering values in the panel when we click next and again swith to 
previous panel,
all its values vanish.
How can we retain those values for a search field?
For rulefield  we  used set="variable set by rule field" 
e.g:

 <field type="rule" variable="db_server">
  <description align="left" txt=""
               id="description.rule.3"/>
  <spec txt="Oracle Server:" set="0:$db_server"
        layout=" AN:10:20" 
        resultFormat="specialSeparator" separator="."/>
  <validator class="com.izforge.izpack.util.NotEmptyValidator"
             txt="The Server Name is mandatory!" />
  <!--processor class=""/-->
</field>


and it worked but what about search and text?
