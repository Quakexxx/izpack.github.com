Hi,

Is there a simple to represent the PacksPanel dependencies visually in a 
hierarchical tree and suppress the dependencies box?

Specifically, I'd like to have it look something like:

Pack1
-Pack2
--Pack3
--Pack4
-Pack5
--Pack6
-Pack7

where the "dashes" represent indentation.

I experimented with inserting spaces in front of the pack names and it
seemed to work fine. As expected, this also requires that all references
to the pack name in izpack.xml, process.xml and izpack-user-input.xml 
need to be updated.  This results in a panel that seems to show the 
hierarchy visually.  Though adding the spaces for each element in the 
xml files is a real hack and does seem troublesome. Is there an easier 
way to accomplish this without unnatural padding or creating a custom 
panel?
