Hi Marc,

Actually on HP-UX df -Pk on local disk gives the correct result

hp1100:51 %v111%df -Pk /scratch
Filesystem          1024-blocks  Used  Available Capacity Mounted on
/dev/vgscr1/lvscr1    35800808 31851688  3949120    89%   /scratch1

But the output of the same command on alpha tru64 unix file system mounted
by autofs gives:

hp1100:52 %v111%df -Pk /euler
Filesystem          1024-blocks  Used  Available Capacity Mounted on
euler:/euler          0        0        0    60%   /tmp_mnt/hosts/euler

while bdf is OK
hp1100:53 %v111%bdf /euler
Filesystem          kbytes    used   avail %used Mounted on
euler:/euler       133160440 78676590 54306664   59% /tmp_mnt/hosts/euler
