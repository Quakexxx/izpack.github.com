A few possibilities :

    * Modify your scripts to redirect all output to log files
    * Wrap your scripts into another that will run them in an xterm of
      some kind.
    * Run your scripts using the ProcessPanel instead of the
      <executable> tag (direct inputs will not be possible, so you'll
      have to use a UserInputPanel first to gather the information, and
      then parse your script with the <parsable> tag to replace the
      variables).

I would advise you to use the latter. It is a bit more complicated, but 
it looks much cleaner to the end-user.

Good luck,
