Hi Domink,
in principle yes..., but you have to use the registry stuff
(see docu under custom actions).
Environment variables on Windows are stored in the registry. There
are two places, one for the user and one generell.

HKCU\Environement
HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\Environment

It is only possible to do it if the installer has the rights to
write into the registry under that paths.
