Hi,

You have to be in the Administrators Usergroup. To have the permission to 
write in the "common" or "all" Users Desktop- or Start Menu folder.

And you need effective write-permission to this folder. In general as Member 
of the Administrators Group you have.

If this also fails, this can be a bug in the native Shelllink.dll - But to be 
sure and for debug reasons give the -DLOG=true option on the commandline.

i.e.~> java -DLOG=true -jar your_installer.jar 

...and send the created logfile to me.

Also important detail: Which language have your WinXP-Box?

Cheers
Marc
