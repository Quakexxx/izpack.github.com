Hi Marcelo,

Which Launcher do you mean, the old wxWidget based Version 1.3?
or the new QT4 based Version 2.0?
For the second (QT4 based) launcher I cant help quickly. I must read the doku 
or sources first, to help.

For the first one, the old wxWidgets based, you have to put a JRE-Entry in the 
[win32]-Section in the launcher.ini

Make shure the setup.exe is real present at this place.
Pathes are relative to the launcher.exe

use the following fragment as template.
##########################
# Sample launcher.ini file.
# Feel free to use it as a basis for your own ones.

# Global entries, can be overriden by specific ones.
jar = IzPack-install.jar
download = http://www.java.com/

# Win32 specific entries
[win32]
jre = jre/setup.exe
...
...
#####################
