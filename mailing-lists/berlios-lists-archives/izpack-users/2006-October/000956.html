Hi,

I agree with Klaus. That would be a huge change for the current App. I have 
for my company a shell script that built three installers (download the new 
elements from a site, extract them to right places, update/delete if 
necessary) each around 300MB  in about 20 minutes... the same batch can be 
used to install or uninstall...
It's fairly easy this way and fast enough.
I'm not saying it's not a good idea, all i'm saying is that it would need 
lots of efforts and time to make it. Especially, i'm not how backward 
compatible it would be with previous install environments...
Might be something interesting to do for a future major release.

Using a shell/batch/Ant script should do the job to start with, i 
believe....
