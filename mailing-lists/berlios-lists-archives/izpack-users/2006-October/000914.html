Hi,
a late answer as clarification (after my holidays):

The message will be only presented as trace info. Do not
declare -DTRACE=TRUE and you will not see the message 
(I hope, else the Debug class will not work correctly).

If you use a custom langpack you should see the message
"Custom langpack for <selected iso3> available."

There is also in the docu a short introduction how to use
custom langpacks. See chapter "Creating Your Own Panels"
and there the paragraph "The Internationalization of custom panels".

Cheers

Klaus
