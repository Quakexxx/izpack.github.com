Hi John,

See the 'loose' attribute of 'pack' elements as explained in the  
documentation:

> loose: can be used so that the files are not located in the  
> installer Jar. The possible values are true or false, the default  
> beeing false. The author of this feature needed to put his  
> application on a CD so that the users could run it directly from  
> this media. However, he also wanted to offer them the possibility  
> to install the software localy. Enabling this feature will make  
> IzPack take the files on disk instead of from the installer. Please  
> make sure that your relative files paths are correct !
