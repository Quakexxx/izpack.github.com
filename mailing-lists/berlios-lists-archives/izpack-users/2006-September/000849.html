Hello all.

I have a question concerning the search field element in the user input
panel of IzPack installer that I could not find an answer to in the
documentation.

As described in the documentation I'd like to let the user specify the
path to the JVM.
So far this works fine, but I have some problems validating the input.
Using the "checkfilename" attribute seems to work only, if the user
really enters something in the field. If it is left blank, pressing the
"Next" button does not show any warning.

It also seems like using the "validator" element within the "search
field" element is not supported. At least the standard
"com.izforge.izpack.util.NotEmptyValidator" did not work.

Do I have to build my own validator to accomplish this or is there
anything else I could do to prevent the user from proceeding with the
installation process without specifying a directory.

Greetings,
