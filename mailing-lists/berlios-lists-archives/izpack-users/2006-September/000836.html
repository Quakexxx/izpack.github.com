>  Hi Jacobo,
> the os attribute declares operating system families. "linux" is near a OS
> name (should be written "Linux"), not a family.
> If you need name dependencies you can use the element "os", but you
> use the attribute (with the
> same name).
> Read docu Writing Installation XML Files->The IZPACKElements -> The Packs
> Element <packs>  -> <file>- add files or directories
> versus
> Writing Installation XML Files->The IZPACKElements -> The Packs Element
> <packs>  -> <os>- make a file OS-dependent
>
> Why "mac" not work I do not know.
